## Subgraph:-

### Install and Setup

#### Clone 

```
git clone https://github.com/RapidInnovation-blockchain/blockchain-solidity-uniswapV3Mock.git

```
 
#### Installation Dependancies

```
cd subgraph

npm install

```

#### Code Generation

``` 
graph codegen

```

#### Graph Build

```
graph build

```

#### Graph Deploy

```
graph deploy --studio <name of subgraph>

```
